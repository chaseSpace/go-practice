## pod

```shell
# 查看
kk get pods -w # watch
# more args:
# -w  等同watch
# -o wide
# --output yaml/json

# 修改
kk replace -f pod.yaml  # 但部分属性无法直接修改，如image等，除非加上 --force参数 重建pod

# 删除
kk delete pod xxx  # 或 -f pod.yaml

# Log
kubectl logs -n kube-system POD_NAME
kubectl logs -f nginx-pod

# Delete
# 强制删除：--grace-period=0 --force
# label过滤：-l k8s-app=calico-node
kubectl delete pod -n kube-system calico-node-pj4h5 

# Describe
kubectl describe pod -n kube-system calico-node-pj4h5

# Exec
kubectl exec -it nginx-pod -- /bin/bash   # 进入pod shell

# Events
kubectl get events -n kube-system
```

## node
```shell
# 删除节点（在master操作）
kubectl delete node k8s-node1 && kubectl get nodes

# 节点再次加入
systemctl stop kubelet && rm -rf /etc/kubernetes/* && rm -rf /etc/cni/net.d
systemctl start kubelet
```


## docker

### docker清理资源

```shell
# 查看可清理的资源
$ docker system df
TYPE            TOTAL     ACTIVE    SIZE      RECLAIMABLE
Images          1         0         4.904MB   4.904MB (100%)
Containers      0         0         0B        0B
Local Volumes   0         0         0B        0B
Build Cache     58        0         622.9MB   622.9MB

# 此命令可以用于清理磁盘，删除关闭的容器、无用的镜像和网络
# 添加 -f 禁用询问
docker system prune
```
